🕒 時程表

5:30交班 (先算櫃台的錢是否正確)，公式：*臨櫃繳費+{收據、團報單} - *臨櫃退費
> 整理教室：
- 1樓櫃台有4個垃圾桶，電梯對面1個
- 2樓：電梯斜對面4個，213、208、203、202各1個
- 廁所走廊有1個垃圾桶
- 廁所要檢查衛生紙與洗手乳
- 每個垃圾桶底下有垃圾袋，缺少請至B1補充
- 檢查教室冷氣、桌面、椅子、抽屜、白板
- 運動教室檢查地板與瑜珈用品

6:10 出發倒垃圾（一定要準時，並向正值報告）
- 1、5 沒有回收車；2、4、6 有回收車
- 星期三沒有垃圾車

9:00 結帳（確認金額正確）
- 公式：*臨櫃繳費+{收據} - *臨櫃退費
- 將交班報表、日報表、現金、{收據}、{退費單} 裝袋交給正值

🏫 教室整理與檢查

- 每天下班前要先將新的點名簿放在架上(除了運動課程皆需要放白板筆)
- 收垃圾、檢查冷氣是否關閉、桌面乾淨、椅子有靠好、抽屜是否有遺留物、白板擦乾淨
- 窗戶需鎖（213、211、210、203、204、206）
- 運動教室使用靜電拖把拖地整理瑜珈墊、瑜珈磚

📝 報名流程

現金繳費：
1. 輸入身分證字號
2. 選課程>折扣（依簡章）
3. 確認報名
4. 臨櫃繳費>收錢
5. 列印收據並蓋章

LINE Pay：
1. 同上流程到確認報名
2. 學友以 LINE Pay 匯款
3. 將臨櫃繳費方式改為 LINE Pay，輸入代碼
4. 臨櫃繳費後列印收據蓋章

轉帳：
1. 同上流程到確認報名
2. 列印繳費單等待入帳
3. 入帳後列印收據蓋章

💰 退費與轉班

- 點選「報名異動」，一律選「退費」

退費流程：
1. 確認原因（如課程未開成）
2. 系統選擇「100%退全額」
3. 點選「臨櫃退費」> 列印退費單
4. 請學友填寫退費資料 > 收回退費單並退還金額
補充說明： 若非停課，依收據背面規定退費，若有疑問請問正值

轉班流程：
轉班流程為一退一進，需要先退費再報名
1. 同上（退費流程）
2. 幫學友報名新課程 > 折扣可以繼承
3. 收補差價或退回多餘款項

👥 團報(不熟悉一定要詢問正值)

紙本處理：
1. 計算正確金額
2. 一般學友85折，10人以上班長7折，否則8折
3. 計算金額公式寫在團報單上
公式 : 一般學友折扣後的費用 * 人數(不含班代) + 班代費用
4. 收取對應的現金>複印團報單蓋章將影本交給學友，因為金額龐大請一定要把錢收對喔

💻 系統入帳
1.確認紙本金額正確
2.先幫班長報名：
    ● 選課、折扣、報名 → 臨櫃繳費 → 印刷收據（如需）
    ● 在紙本團報單班長欄位打勾
3其他學友報名：
    ● 入資料→課程代號、折扣代號 → + →確認報名 → 臨櫃繳費 → 印刷收據
    ● 重複至全部報名完畢
⚠️只有已繳費的學友才能進入系統
